Titulo princal - h1

O h1 é utilizado para colocar o titulo principal da página, o assunto/tema principal que será abordado

Subtitulos - h2

Diferente do h1, o h2 é utilizado para Titulos de seções principais

Subseções - h3

O h3 é utilizado para subseções dentro de um h2, onde você pode separar uma seção em algumas partes menores

Detalhes adicionais - h4

O h4 é utilizado para detalhes adicionais , dentro de uma subseção quando a necessitade de criar uma parte mas ainda dentro da sua subseção para explicar algo

Especificar mais - h5

Ele tem a mesma função do h4, porém é menor , então é utilizado para especificar mais ainda.
